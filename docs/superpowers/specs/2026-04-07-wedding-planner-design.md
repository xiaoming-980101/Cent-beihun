# 婚礼筹备助手 - 设计文档

> 基于 Cent-beihun 项目扩展，增加婚礼筹备全流程管理功能

## 项目概述

### 基本信息

| 项目 | 内容 |
|------|------|
| 项目名称 | 婚礼筹备助手（Cent-beihun 扩展版） |
| 使用场景 | 个人使用（情侣协同） |
| 技术栈 | React 19 + Vite + TypeScript + Tailwind CSS |
| 数据存储 | Gitee（国内访问优化） |
| 跨平台支持 | PWA（PC端 + 移动端） |

### 时间线

| 事件 | 日期 | 备注 |
|------|------|------|
| 订婚 | 2026年5月22日 | 约45天后 |
| 结婚 | 2027年5月15日 | 约13个月后 |

---

## 整体架构

### 应用结构

```
底部导航（5个Tab）
├── 首页（Home）        - 倒计时、概览、快捷入口
├── 记账（Ledger）      - 现有记账功能（保留）
├── 任务（Tasks）       - 任务管理、日历视图、进度跟踪
├── 工具（Tools）       - 礼金簿、座位表、亲友管理、吉日推荐、回忆时光轴
└── 设置（Settings）    - 现有设置功能（保留）
```

### 数据存储结构

复用 Cent 的 GitHub/Gitee 存储机制，扩展婚礼专属数据：

```typescript
// 数据存储在 Gitee 仓库的 JSON 文件中
type AppData = {
  // 现有数据（保留）
  items: Bill[];              // 账单记录
  meta: {
    tags: BillTag[];
    categories: BillCategory[];
    budgets: Budget[];
    customCurrencies?: CustomCurrency[];
    baseCurrency?: string;

    // 新增婚礼数据
    wedding: WeddingData;
  }
}

type WeddingData = {
  // 基础信息
  engagementDate: number;      // 订婚日期时间戳
  weddingDate: number;         // 结婚日期时间戳

  // 任务管理
  tasks: WeddingTask[];

  // 亲友管理
  guests: Guest[];

  // 礼金簿
  giftRecords: GiftRecord[];

  // 座位表

  // 回忆时光轴

  favoredDates: number[];

```

type WeddingTask = {

## 开发优先级

考虑到订婚日期较近（45天），建议按以下优先级开发：

### 第一阶段：核心功能（订婚前完成）

| 优先级 | 功能 | 原因 |
|--------|------|------|
| P0 | 婚礼倒计时 | 订婚必备 |
| P0 | 礼金簿 | 订婚需要记录礼金 |
| P0 | 亲友管理 | 礼金簿依赖 |
| P1 | 任务系统基础版 | 备婚规划 |
| P1 | 预算扩展（定金/尾款） | 财务管理 |

### 第二阶段：完善功能（订婚后、结婚前）

| 优先级 | 功能 | 原因 |
|--------|------|------|
| P1 | 座位表 | 结婚宴会需要 |
| P1 | 日历视图 | 任务管理增强 |
| P2 | 吉日推荐 | 锦上添花 |
| P2 | 回忆时光轴 | 记录美好瞬间 |
| P2 | 任务系统完整版 | 15大分类模板 |
