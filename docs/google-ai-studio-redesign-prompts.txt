# Google AI Studio 对比生图提示词

本文档用于在 Google AI Studio 中生成 Cent / YueWed 项目的页面重设计图，方便与 `chatgpt-images-2` 输出做横向对比。

使用方式：

1. 先复制“全局项目设定”。
2. 再复制“白天/黑夜基础色调”。
3. 最后追加对应页面的“页面任务提示词”。
4. 每个页面至少生成白天版和黑夜版；涉及表单、弹窗、筛选、详情、确认、空状态的页面，额外生成覆盖层状态。

## 与当前项目的关系

这些提示词不是把 Cent 改成一个全新的婚礼 App，而是在现有 Cent 记账 PWA 上做 YueWed 婚礼主题重设计。当前项目有两层产品身份，生成图时必须同时保留：

```text
Layer 1: Original Cent finance app capabilities
- collaborative ledger/books
- bill add, edit, detail, delete
- central add button with normal add, voice entry, keyboard quick entry
- search and advanced filters
- analytics/statistics
- category and tag management
- multi-currency and exchange rates
- scheduled/recurring bills
- original budget management
- data import/export, backup, Smart Import, import preview
- GitHub, Gitee, WebDAV, S3, offline sync endpoints
- configurable AI assistant and chat cards
- theme, language, map, lab, about, user profile settings

Layer 2: YueWed wedding extension
- wedding home dashboard
- wedding tasks and task calendar
- gift ledger
- guest management
- wedding vendor budget
- planning suggestions
- resource center
```

Hard rule:

```text
Do not remove or hide the original Cent identity. The UI should look like a wedding redesign of a finance PWA, not a pure wedding planning app.
```

## 全局项目设定

```text
Create a high-fidelity mobile app UI screenshot for YueWed / Cent, a wedding planning and personal finance PWA.

The product is not a marketing website. It is a real daily-use productivity app for couples and families planning a wedding. It combines wedding tasks, guest management, gift ledger, budget tracking, financial statistics, search, AI planning suggestions, settings, and data backup.

Important product fidelity: this is still the existing Cent finance PWA. Keep visible or implied entry points for bill creation, ledger/books, sync, search, statistics, data manager, AI assistant, category/tag/currency/scheduled bill settings, user profile, theme, and language. Do not simplify it into only wedding tasks and guests.

Visual direction:
- modern, refined, emotional but practical
- premium wedding planning tool, not a generic pink app
- clear information hierarchy, real product density, easy to scan
- inspired by Apple Health, Linear, Craft, Structured, and a high-end wedding planning notebook
- Chinese UI text
- mobile app screenshot, 390x844
- realistic app interface, not poster, not landing page, not Figma frame

Component style:
- clean top app bar
- rounded cards with 16-28px radius
- subtle shadows, thin borders, soft surfaces
- line icons similar to lucide or Material Design Icons
- readable charts, tags, buttons, lists, forms
- fixed bottom action area for dialogs and forms
- avoid decorative clutter

Do not:
- do not make it a marketing hero page
- do not use cartoon wedding illustrations
- do not fill the whole screen with cheap pink gradients
- do not add explanatory design notes inside the UI
- do not make controls look fake or non-clickable
```

## 白天基础色调

```text
Day theme base palette:
- App background: #FAFAFA, #FFF7FB, #F8FAFC
- Main cards: #FFFFFF, with very subtle pink/lavender surfaces #FFF7FB -> #F8F3FF
- Primary brand: #F472B6, #EC4899
- Secondary: #A855F7, #3B82F6
- Success: #22C55E
- Warning: #F97316
- Danger: #EF4444
- Main text: #111827
- Body text: #374151
- Muted text: #6B7280, #9CA3AF
- Borders: #E5E7EB, #F3E8FF
- Gold may appear only as a thin accent line or tiny icon detail, not a main color.

The day theme should feel like warm white paper, misty rose, ice blue function color, and restrained lavender ceremony.
```

## 弹窗与覆盖层通用提示词

```text
Create the overlay/dialog state for the same YueWed / Cent mobile app page.

Requirements:
- 390x844 mobile screenshot
- keep the original page visible behind a translucent blurred or dimmed overlay
- use a bottom sheet for simple dialogs and filters
- use a full-screen mobile sheet for complex forms
- include a drag handle, title, description, close/back control
- form fields must look real and filled with useful Chinese sample data
- include fixed bottom actions: cancel and confirm
- confirm button uses the page accent color or the pink-purple brand gradient
- do not let the bottom actions cover the form content
- generate both day theme and night theme versions
```

## 对比评审维度

用 Google AI Studio 生成后，和 `chatgpt-images-2` 对比时重点看：

- 页面是否像真实 App，而不是概念图。
- 中文是否自然、信息是否可读。
- 白天/黑夜是否同一套设计语言。
- 弹窗是否能指导实际代码复刻。
- 表单字段是否完整，是否覆盖页面真实交互。
- 统计、搜索、预算这类高密度页面是否仍然清楚。
- 是否有过度粉色、过度霓虹、过度卡通的问题。
