## 执行协议

### 代码修改原则
1. **仅修改样式** - 不触碰业务逻辑代码
2. **使用 Tailwind 类** - 优先使用 Tailwind 类名
3. **CSS 变量优先** - 颜色、间距等使用 CSS 变量
4. **保留功能代码** - 任何 onClick、onChange 等事件处理保持不变

### 子代理调用规则
- 使用 `executor` 子代理执行具体页面改造
- 子代理提示词末尾附加 `$vibe`
- 子代理继承需求文档和设计规范

### 验证协议
- 每波完成后运行 `npm run dev` 目视验证
- 发现问题立即回滚并重新设计
- 最终验收需截图对比设计稿
